# Web3 Whale Activity Tracker

A professional Web3 reference project that demonstrates how large wallet activity and high-impact blockchain transfers can be modeled using clean and modular Web3 architecture.

This repository focuses on simulating whale transactions, monitoring large token movements, detecting market-impact events, and generating analytics reports in a beginner-friendly and production-style format.

---

## Supported Networks

- Ethereum  
- Base  
- Polygon  
- Arbitrum  

---

## Project Goals

- Simulate whale wallet transactions  
- Track large token transfers  
- Detect market-impact events  
- Analyze transaction patterns  
- Teach on-chain monitoring architecture  

---

## Architecture Overview

This project uses a root-only structure where each file handles a single responsibility such as network access, wallet registry, transfer engine, impact analyzer, and analytics reporting.

---

## Features

- Multi-chain RPC support  
- Whale transaction simulation engine  
- Market impact detection logic  
- Transfer analytics reporting  
- Clean console-based output
